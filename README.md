# dns320_utils

## required libraries
none

## compilation

make 

## simple commands:

* Power off led
  `./dns320l-cli /dev/ttyS1 PwrLedOnCmd ` 

* Power on led
  `./dns320l-cli /dev/ttyS1 PwrLedOffCmd `

* Read the alarm 
  `./dns320l-cli /dev/ttyS1 ralarm`
